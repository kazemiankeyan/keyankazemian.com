What I'm currently working on.

lo-key is currently under construction...

In a nutshell, lo-key is an application that helps shed light on smaller and more undiscovered artists. Using the Spotify Web Api, Lo-Key allows the user to input their favorite artist and receive a list of similar yet "low key" artists.

It is my strong belief that in today's society, the majority of the population listens to a select number of regurgitated musicians. Unfortunately, this means that there is a plethora of popular artists of yesterday and unpopular artists of today that are constantly being slept on.

lo-key hopes to change that.
